# frontend/

React + TypeScript + Vite frontend application.

## Purpose
Complete frontend web application for Mastermind game built with React, TypeScript, Vite, and Tailwind CSS. Provides interactive UI for playing the game.

## Contents

### Subdirectories

- **src/** - Application source code (components, store, services, types)
- **public/** - Static assets served directly
- **node_modules/** - NPM dependencies (excluded from documentation)

### Files

- **package.json** - NPM dependencies and scripts
- **package-lock.json** - Locked dependency versions
- **vite.config.ts** - Vite build configuration with API proxy
- **tsconfig.json** - TypeScript configuration root
- **tsconfig.app.json** - TypeScript config for application code
- **tsconfig.node.json** - TypeScript config for Node.js code
- **tailwind.config.js** - Tailwind CSS configuration
- **postcss.config.js** - PostCSS plugins (Tailwind + Autoprefixer)
- **eslint.config.js** - ESLint configuration for code quality
- **index.html** - HTML entry point
- **README.md** - Frontend documentation
- **CLAUDE.md** - Frontend development guidelines

None (configuration files only)
